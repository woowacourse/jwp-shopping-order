# jwp-shopping-order

## 기능 구현 목록
- [ ] 장바구니의 상품 중 주문 완료된 상품은 장바구니에서 제거한다.
- [ ] 주문 금액에 따라 할인이 적용된다.
    - 5만원 이상 주문 시 2000원 할인
    - 10만원 이상 주문 시 5000원 할인
    - 20만원 이상 주문 시 12000원 할인
- [ ] 주문 내역을 조회한다.
   - [ ] 상세 정보 조회
       - 주문 번호
       - 각 품목 정보 - 품목명, 수량, 가격
       - 주문 날짜
       - 할인 전 금액
       - 전체 할인 금액
       - 할인 후 금액
   - [ ] 전체 목록 조회
       - 주문 번호
       - 총 주문 금액
       - 품목명
       - 주문 날짜
