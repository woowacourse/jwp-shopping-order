=== 개요

=== 상태코드 (HttpStatus)
응답시 다음과 같은 응답상태를 제공합니다.

[cols="3,3"]
|====
|HttpStatus |설명

|`OK(200)` |정상 응답
|`UNAUTHORIZED(401)` |로그인되지 않은 상황에서 요청할 때 발생합니다.
|`FORBIDDEN(403)` |권한이 없는 요청을 할 때 발생합니다.

|====
